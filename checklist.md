# Acceptance Checklist

## Design & UI
- [ ] The site uses a dark background and monospace fonts globally.
- [ ] Navigation mimics a command-line prompt or terminal interface.
- [ ] Text is legible with sufficient contrast.
- [ ] Mobile view is usable (no horizontal scrolling on text, menus accessible).

## Features
- [ ] **Home Page**:
    - [ ] Displays the "Typing Effect" introduction.
    - [ ] Shows personal information/bio.
    - [ ] Links to GitHub/Email work.
- [ ] **Blog**:
    - [ ] Posts are listed in reverse chronological order.
    - [ ] Code blocks in posts have syntax highlighting.
    - [ ] Images in posts render correctly.
- [ ] **Tech**:
    - [ ] Jekyll build succeeds locally (simulated) and on GitHub Pages.
    - [ ] No console errors in the browser.

## Content
- [ ] Includes at least one sample post.
- [ ] `_config.yml` has the correct user details (buaawzz).
